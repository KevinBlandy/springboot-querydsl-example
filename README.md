
# SpringBoot-QueryDsl-Example

## 软件版本

- SpringBoot 2.6.1
- Java 17
- MYSQL 8.x

> 需要手动创建数据库，系统启动会后自动创建数据表（包括索引）。

## Example代码

都在 `src/main/resources` 目录下


- `DataInit` 初始化演示数据（最先执行）
- `Example1` 单表的查询/编辑/删除
- `Example2` join查询
- `Example3` 分页/排序

- 子查询
- exists查询
- 聚合查询
- 条件分组
- 加锁
- 结果集封装
- spring-data-jpa 的支持
